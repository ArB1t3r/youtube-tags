Privacy Policy

Last updated: April 24, 2026

YouTube Tags is a browser extension that helps you organize YouTube subscriptions with custom tags. Your privacy is important to us.

Data We Collect

YouTube Tags does not collect, transmit, or sell any personal data to third parties. All data stays on your device by default.

The extension stores the following data locally in your browser using the Chrome Storage API:

  • YouTube channel names, handles, and thumbnails detected from pages you visit on youtube.com
  • Tags you create and their assignments to channels
  • Watch counts and viewing history (video IDs only, used for sidebar sorting)
  • Your UI preferences (filter states, sidebar mode, content filter toggles)

Optional Third-Party Services

The following integrations are entirely optional and only activated when you explicitly configure them in Settings:

  • AI Tag Suggestions (OpenAI / Anthropic) — If you provide an API key, channel names and your tag list are sent to the selected AI provider to generate tag suggestions. Your API key is stored locally and never shared. Refer to OpenAI's or Anthropic's privacy policies for how they handle API requests.
  • Supabase Sync — If you configure Supabase credentials, your tags and channel data are synced to your own Supabase project for cross-browser persistence. You control the database — we have no access to it.

Permissions

  • Host permission (youtube.com) — Required to read channel information from YouTube pages and inject the tag UI.
  • Storage — Required to save your tags, channels, and preferences locally.
  • Tabs — Required to open the options and welcome pages.
  • Alarms — Required for periodic Supabase sync (if configured).

Data Deletion

You can delete all extension data at any time by:

  • Uninstalling the extension (removes all local data)
  • Clearing site data for the extension in Chrome settings
  • Using the "Reset All Watch Counts" button in Settings to clear watch history

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
